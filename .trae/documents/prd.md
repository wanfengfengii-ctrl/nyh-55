## 1. 产品概述

差分机模拟器是一个基于 Web 的交互式应用，用于模拟巴贝奇早期机械差分机的数字轮、进位杆和齿轮联动过程。用户可以设置初始数列、差分阶数和手柄转动次数，通过动画观察数字轮转动、进位触发和结果输出，深入理解差分机的机械运算原理。

- 目标用户：数学爱好者、计算机科学教育者、机械计算历史研究者
- 核心价值：将抽象的差分运算转化为可视化的机械运动过程，提供沉浸式学习体验

## 2. 核心功能

### 2.1 功能模块

1. **主页面**：差分机机械动画画布、参数控制面板、运算记录面板

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | 机械动画画布 | PixiJS 渲染数字轮、进位杆、齿轮联动的机械动画 |
| 主页面 | 参数设置面板 | 设置初始数列、差分阶数、手柄转动次数、模数（基数） |
| 主页面 | 操作控制栏 | 单步执行、连续执行、回退、重置按钮 |
| 主页面 | 差分表格 | 实时显示差分表格的数值变化 |
| 主页面 | 运算记录面板 | 完整的运算步骤日志，包括每步操作类型、数值变化 |
| 主页面 | 错误状态提示 | 非法状态时高亮显示错误进位，禁止继续运算 |

## 3. 核心流程

用户打开应用 → 设置初始数列和差分阶数 → 点击单步/连续执行 → 观察数字轮转动动画 → 进位触发时齿轮联动 → 查看差分表格更新 → 遇到非法状态则高亮报错 → 可回退到上一步 → 运算完成后查看完整记录

```mermaid
    "A[用户设置参数]" --> "B[初始化差分机]"
    "B" --> "C{选择操作}"

    "J" --> "K{是否触发进位?}"
    "K" --> "是" --> "L[播放进位齿轮联动]"
    "K" --> "否" --> "M[更新差分表格]"
    "L" --> "M"
    "M" --> "N{是否非法状态?}"
    "N" --> "是" --> "O[高亮错误进位，禁止继续]"

    "Q" --> "是" --> "R[输出最终结果]"
    "F" --> "S[恢复上一步机械状态]"
    "S" --> "C"
    "G" --> "A"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主题**：维多利亚工业/蒸汽朋克风格，致敬巴贝奇差分机的历史背景
- **主色调**：黄铜色（#C8A951）、深青铜色（#4A3728）、氧化铜绿（#2E8B57）
- **辅助色**：羊皮纸白（#F5F0E1）、墨水黑（#1A1A2E）、警示红（#C0392B）
- **字体**：展示字体使用 Playfair Display（衬线体，维多利亚气质），UI字体使用 Source Sans 3
- **按钮风格**：圆润金属质感，带有微妙的光泽和阴影
- **布局风格**：左侧大画布（机械动画），右侧控制面板和记录
- **图标风格**：线性齿轮、扳手等机械图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 主页面 | 机械动画画布 | 深色背景，黄铜色数字轮，铜绿色齿轮，进位杆为金属质感，带微妙光晕 |
| 主页面 | 参数设置面板 | 羊皮纸色背景卡片，金属边框，输入框带铜色描边，下拉选择器 |
| 主页面 | 操作控制栏 | 金属质感圆形按钮，执行按钮为铜绿色，回退按钮为黄铜色，重置为暗红色 |
| 主页面 | 差分表格 | 表格式布局，当前步高亮，进位行特殊标记，错误行红色背景 |
| 主页面 | 运算记录面板 | 滚动列表，每步带时间戳、操作类型图标、数值变化对比 |
| 主页面 | 错误状态提示 | 红色闪烁边框，错误齿轮高亮，禁止操作的灰色遮罩 |

### 4.3 响应式设计

- 桌面端优先（1920x1080 为基准）
- 画布区域自适应宽度，最小宽度 600px
- 控制面板在窄屏下折叠为底部抽屉
- 数字轮大小随画布缩放

### 4.4 动画设计

- **数字轮转动**：0→9 循环滚动，平滑旋转动画，带有惯性缓动
- **进位触发**：进位杆从静止位推动到触发位，相邻齿轮同步旋转
- **齿轮联动**：主齿轮旋转带动从动齿轮反向旋转，传动比可视化
- **错误高亮**：错误部件红色脉冲发光，2秒闪烁周期
- **回退动画**：数字轮反转回到前一状态
